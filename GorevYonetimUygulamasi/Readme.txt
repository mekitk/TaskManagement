## 🎯 Görev Yönetim Sistemi (Task Manager)
Tam özellikli proje/görev yönetimi uygulaması.

### 🚀 Canlı Demo
[https://taskmanager-client.vercel.app](https://taskmanager-client.vercel.app)

### 🛠 Kullanılan Teknolojiler
- Backend: ASP.NET Core Web API, MongoDB
- Frontend: Next.js, Tailwind CSS, Redux
- Auth: JWT
- Gerçek Zamanlı Bildirim: SignalR (opsiyonel)
- Test: xUnit (opsiyonel)

### 📁 Proje Dizini
- /controllers
- /routes
- /services
- /models
- /middlewares

### 🔐 Roller
- Admin
- Manager
- Developer

### ➕ Ekstra Özellikler
- Görev log geçmişi (MongoDB Log koleksiyonu)
- WebSocket ile bildirim (Bonus)
